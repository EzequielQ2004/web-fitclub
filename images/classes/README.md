# Imágenes de Clases - FitClub

## Estructura de Archivos
Las imágenes de las clases deben colocarse en la carpeta `images/classes/` con los siguientes nombres exactos:

- `funcional-musculacion.jpg` - Para la clase de Funcional/Musculación
- `crossfit.jpg` - Para la clase de Crossfit
- `adultos-mayores.jpg` - Para la clase de Actividad Física Adultos Mayores
- `danza-aerea.jpg` - Para la clase de Danza Aérea (Tela)
- `taekwondo.jpg` - Para la clase de Taekwondo

## Pasos para Actualizar las Imágenes:

1. **Obtén las imágenes reales** de cada clase del gimnasio
2. **Redimensiona las imágenes** (recomendado: 800x600px o similar)
3. **Nombra los archivos** exactamente como se indica arriba
4. **Reemplaza los archivos** en la carpeta `images/classes/`
5. **Elimina los archivos .txt** una vez que hayas agregado las imágenes reales

## Formato Recomendado:
- **Formato:** JPG o PNG
- **Tamaño:** 800x600px (mínimo)
- **Peso:** Menos de 500KB por imagen
- **Calidad:** Alta pero optimizada para web

## Notas:
- Las imágenes deben mostrar claramente la actividad correspondiente
- Se recomienda usar imágenes con buena iluminación
- Si es posible, incluir personas realizando la actividad
- Mantener la coherencia visual con la marca FitClub

Una vez reemplazados los archivos, el sitio web mostrará automáticamente las imágenes reales.
